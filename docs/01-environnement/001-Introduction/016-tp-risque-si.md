# TP : Les risques d'un SI

**TP 20 minutes** : Trouver les mesures de prévention associées à ces risques

*   **Ransomware as a Service (RaaS) :** Services de rançongiciels proposés aux cybercriminels3.
*   **Accès non autorisé:** Accès à des systèmes ou des données sans autorisation.
*   **Phishing :** Technique d'obtention d'informations confidentielles en se faisant passer pour une entité de confiance.
*   **Malwares :** Logiciels malveillants incluant les logiciels espions.
*   **Vulnérabilités non corrigées :** Systèmes non mis à jour laissant des failles de sécurité exploitables1.
*   **Mots de passe faibles :** Utilisation de mots de passe faciles à deviner ou à craquer.
*   **Attaques par déni de service (DoS) :** Tentatives de rendre un service indisponible.
*   **Cryptojacking :** Utilisation non autorisée des ressources informatiques pour miner des cryptomonnaies.
*   **Attaques via les objets connectés (IoT) :** Exploitation des failles de sécurité des appareils connectés.
*   **Injection SQL :** Technique d'exploitation des failles dans les bases de données.
*   **Zero-Day Exploit :** Exploitation de vulnérabilités inconnues avant qu'elles ne soient corrigées.
*   **Erreurs humaines :** Négligence ou ignorance du personnel conduisant à des incidents de sécurité.
*   **Vol de données :** Accès non autorisé et extraction d'informations sensibles.
*   **Défaillances matérielles :** Pannes d'équipements entraînant des pertes de données ou des interruptions de service.
*   **Manque de mises à jour et de maintenance :** Absence de correctifs de sécurité et d'entretien régulier des systèmes.

# Solution 


| Risque                                    | Mesures de Prévention                                                                                                                                                                                                                                        |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ransomware as a Service (RaaS)            | - Solutions de détection et de réponse aux menaces tests de sécurité réguliers                                                                                                                                                                                                   |
|                                           | - Sauvegardes régulières et tests de restauration.                                                                                                                                                                                                                |
|                                           | - Sensibilisation des utilisateurs aux risques de phishing et de téléchargement de fichiers suspects.                                                                                                                                                            |
| Accès non autorisé                          | - Authentification forte (multi-facteur).                                                                                                                                                                                                                         |
|                                           | - Contrôle d'accès basé sur les rôles (RBAC).                                                                                                                                                                                                                       |
|                                           | - Surveillance et audit des accès.                                                                                                                                                                                                                              |
| Phishing                                  | - Formation des utilisateurs à la reconnaissance des tentatives de phishing.                                                                                                                                                                                          |
|                                           | - Filtres anti-spam et anti-phishing.                                                                                                                                                                                                                          |
|                                           | - Simulation de phishing pour tester la vigilance des utilisateurs.                                                                                                                                                                                            |
| Malwares (incluant les logiciels espions) | - Antivirus et anti-malware à jour.                                                                                                                                                                                                                             |
|                                           | - Analyse comportementale pour détecter les activités suspectes.                                                                                                                                                                                                |
|                                           | - Sandboxing pour l'exécution de fichiers suspects dans un environnement isolé.                                                                                                                                                                                   |
| Vulnérabilités non corrigées               | - Gestion des correctifs (patch management) rigoureuse.                                                                                                                                                                                                                |
|                                           | - Scans réguliers de vulnérabilités.                                                                                                                                                                                                                           |
|                                           | - Utilisation d'un système de gestion des vulnérabilités (Vulnerability Management System).                                                                                                                                                                  |
| Mots de passe faibles                      | - Politiques de mots de passe forts (longueur, complexité, renouvellement).                                                                                                                                                                                         |
|                                           | - Authentification multi-facteur (MFA).                                                                                                                                                                                                                         |
|                                           | - Gestionnaire de mots de passe.                                                                                                                                                                                                                                 |
| Attaques par déni de service (DoS)        | - Pare-feu applicatif web (WAF).                                                                                                                                                                                                                                 |
|                                           | - Solutions de mitigation DDoS (basées sur le cloud ou sur site).                                                                                                                                                                                             |
|                                           | - Limite de débit et filtrage du trafic.                                                                                                                                                                                                                         |
| Cryptojacking                              | - Détection des processus de minage de cryptomonnaies.                                                                                                                                                                                                                   |
|                                           | - Blocage des sites web et des extensions de navigateur utilisés pour le cryptojacking.                                                                                                                                                                        |
|                                           | - Surveillance de l'utilisation des ressources système (CPU, mémoire).                                                                                                                                                                                           |
| Attaques via les objets connectés (IoT)  | - Segmentation du réseau pour isoler les appareils IoT.                                                                                                                                                                                                                |
|                                           | - Mise à jour régulière des firmwares des appareils IoT.                                                                                                                                                                                                          |
|                                           | - Modification des mots de passe par défaut des appareils IoT.                                                                                                                                                                                                |
| Injection SQL                              | - Validation et nettoyage des entrées utilisateur.                                                                                                                                                                                                                    |
|                                           | - Utilisation de requêtes paramétrées ou de procédures stockées.                                                                                                                                                                                                     |
|                                           | - Pare-feu applicatif web (WAF) avec des règles de protection contre les injections SQL.                                                                                                                                                                           |
| Zero-Day Exploit                           | - Détection et réponse aux intrusions (IDS/IPS).                                                                                                                                                                                                                       |
|                                           | - Analyse comportementale pour identifier les activités anormales.                                                                                                                                                                                                |
|                                           | - Solutions de sandboxing pour l'exécution de fichiers suspects.                                                                                                                                                                                                  |
| Erreurs humaines                            | - Formation et sensibilisation du personnel aux risques de sécurité.                                                                                                                                                                                                |
|                                           | - Politiques et procédures de sécurité claires et documentées.                                                                                                                                                                                                    |
